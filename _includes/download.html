CernVM-FS Package Repositories

yum

https://cvmrepo.s3-website.cern.ch/cvmrepo/yum/

In order to add the yum repository, run: 

sudo yum install https://cvmrepo.web.cern.ch/cvmrepo/yum/cvmfs-release-latest.noarch.rpm

apt

https://cvmrepo.s3-website.cern.ch/cvmrepo/apt

In order to add the apt repository, run: 


wget https://cvmrepo.web.cern.ch/cvmrepo/apt/cvmfs-release-latest_all.deb
sudo dpkg -i cvmfs-release-latest_all.deb
rm -f cvmfs-release-latest_all.deb
sudo apt-get update
